Phase 1 — Core task engine
Projects
POST
/projects
নতুন project create করো
body: name, key, description
GET
/projects
আমার সব projects list করো (member হিসেবে)
GET
/projects/:projectId
একটা project এর detail দেখো
PATCH
/projects/:projectId
Project এর name/description update করো
admin only
DELETE
/projects/:projectId
Project archive / soft delete
admin only
POST
/projects/:projectId/members
Project এ member add করো
body: userId, role (admin|member)
DELETE
/projects/:projectId/members/:userId
Member কে project থেকে remove করো
GET
/projects/:projectId/members
Project এর সব member দেখো
Tasks / Issues
POST
/projects/:projectId/tasks
নতুন task create করো
body: title, description, priority, assigneeId, dueDate, sprintId?
GET
/projects/:projectId/tasks
Project এর সব task list (filter + pagination)
query: status, assigneeId, priority, sprintId, page, limit
GET
/projects/:projectId/tasks/:taskId
একটা task এর full detail
PATCH
/projects/:projectId/tasks/:taskId
Task update করো (title, desc, priority, dueDate)
PATCH
/projects/:projectId/tasks/:taskId/status
Task এর status পরিবর্তন করো
body: status (todo|in_progress|review|done)
PATCH
/projects/:projectId/tasks/:taskId/assign
Task কাউকে assign করো বা reassign
body: assigneeId
DELETE
/projects/:projectId/tasks/:taskId
Task delete করো
POST
/projects/:projectId/tasks/:taskId/subtasks
Sub-task create করো (optional)
parentId দিয়ে task এর নিচে task
Sprints
POST
/projects/:projectId/sprints
নতুন sprint create করো
body: name, startDate, endDate
GET
/projects/:projectId/sprints
Project এর সব sprint list
GET
/projects/:projectId/sprints/active
Currently active sprint
PATCH
/projects/:projectId/sprints/:sprintId
Sprint update (name, dates)
PATCH
/projects/:projectId/sprints/:sprintId/start
Sprint শুরু করো (status: active)
PATCH
/projects/:projectId/sprints/:sprintId/complete
Sprint শেষ করো (status: completed)
POST
/projects/:projectId/sprints/:sprintId/tasks
Backlog থেকে sprint এ task move করো
body: taskIds[]
DELETE
/projects/:projectId/sprints/:sprintId/tasks/:taskId
Sprint থেকে task সরিয়ে backlog এ ফেলো
Phase 2 — Collaboration
Comments
POST
/tasks/:taskId/comments
Task এ comment করো
body: content, parentId? (reply হলে)
GET
/tasks/:taskId/comments
Task এর সব comment (nested)
PATCH
/tasks/:taskId/comments/:commentId
নিজের comment edit করো
DELETE
/tasks/:taskId/comments/:commentId
Comment delete করো
owner বা admin
Attachments
POST
/tasks/:taskId/attachments
Task এ file attach করো
multipart/form-data, Multer
GET
/tasks/:taskId/attachments
Task এর সব attachment list
DELETE
/tasks/:taskId/attachments/:attachmentId
Attachment delete করো
Activity log
GET
/tasks/:taskId/activity
Task এর সব change history
auto-generated, manual endpoint নেই
GET
/projects/:projectId/activity
Project-level সব activity feed
query: page, limit
Phase 3 — Power features
Labels / Tags
POST
/projects/:projectId/labels
Project এ নতুন label তৈরি করো
body: name, color (#hex)
GET
/projects/:projectId/labels
Project এর সব label
PATCH
/projects/:projectId/labels/:labelId
Label update (name, color)
DELETE
/projects/:projectId/labels/:labelId
Label delete করো
POST
/tasks/:taskId/labels
Task এ label attach করো
body: labelIds[]
DELETE
/tasks/:taskId/labels/:labelId
Task থেকে label সরাও
Notifications
GET
/notifications
আমার সব notification (unread first)
query: isRead, page, limit
PATCH
/notifications/:notificationId/read
একটা notification read mark করো
PATCH
/notifications/read-all
সব notification read করো
GET
/notifications/unread-count
Unread count (badge এর জন্য)
Search & Filter
GET
/projects/:projectId/tasks/search
Task search করো
query: q (full-text), status, priority, assigneeId, labelId, sprintId, dueBefore, dueAfter
GET
/search
Global search (tasks + projects)
query: q, type (task|project)
